Basic Info on Kataform

Kataform is being coded in go. It is using the graphics system of GoGi. The syntax of go will be talked about on later slides, but right now we will look at the syntax of GoGi.

In each file of code, there will be something called the trow. This is what you will add everything to. The following code creates a new text element inside of the trow and then sets the text to Hello World.

text1 := trow.AddNewChild(gi.KiT_Label, "text1").(*gi.Label)
text1.Text = "Hello World"

The first line of code creates the text and the second line of code sets the text. This syntax can be used in a lot of cases, but most of the time you will just be copying this on different projects and do not need to master it.

Game -- Battle Game on Kataform

This project is a battle game, currently in its working stages. Right now it is being worked on to create the Map, using a for loop setup. Here are a few things that you will want to do:

Adding objects to the Map

To add an object to the Map, add the following code:

Map = append(Map, mapRow{"elementName", "elementType", 7, 8, 9})

In that code, where it says elementName, enter what you want to be the element's name. Where it says elementType, enter the element type. More on types later. Where it says 7, write the x position, where it says 8 enter the y position and where it says 9 enter the z position. This line of code can be copied throughout the code.

Adding types

Types are different blocks or groups of blocks to be added to the map. grassBlock could be a type. Inside of your code file, there should be a for loop with if statments inside of it. The start of this code should look something like this:

for i := 0; i < len(Map); i++ {
if Map[i].typeOf == "grassBlock" {
some code in here ...
}

After that end bracket after the code, type the following:

else if Map[i].typeOf == "newType" {
some code here ...
}

Where it says newType, type in the name of the new type. Then copy the code inside the if statement from another block and put it inside the new else if statement. Change any place where it says the name of another type and put your new type instead. Where it says something dot jpg, like path.jpg, go to google and find an image of your type. Save the image in your game folder as typeName.jpg, like dirt.jpg. Then in the code, set the image name to the thing you saved it as. Add an object to the map with your new type, save the code and run it!
